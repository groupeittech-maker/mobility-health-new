# Arrière-plan carte d'assurance

Pour afficher une carte de l'Afrique en arrière-plan de la carte d'assurance voyage, placez une image nommée **`africa-map.png`** dans ce dossier (`frontend-simple/assets/`).

- Format conseillé : PNG avec fond transparent (silhouette du continent).
- Si le fichier est absent, la carte utilise un fond violet uni sans motif.

Vous pouvez utiliser une silhouette du continent Afrique en domaine public (par ex. Public Domain Vectors, PikWizard, CleanPNG).
